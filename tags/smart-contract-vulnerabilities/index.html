Solidity Vulnerabilities

2022-06-08Somesh Banerjee
#Solidity  #Solidity-Vulnerabilities  #Smart-Contract-Vulnerabilities 
Introduction Smart contracts built on blockchain platforms like Ethereum have transformed various industries by enabling decentralized and trustless applications. Smart contracts are written in solidity language and deployed in the EVM blockchain, which enable self-executing agreements without the need for intermediaries, revolutionizing industries such as finance, governance, and others. However, we know blockchain is immutable and so are smart contracts; Once they are deployed, it cannot be changed. This means that any security flaw or vulnerability within the code remains permanently exploitable.
